Here are some of my projects

  • Stack Overflow Clone

    Html, CSS, Javascript, ReactJS, Node.js, MongoDB, Express

    Full-Stack website cloning the popular website stackoverflow.com. This project utilizes the MERN technology stack to create a database for the questions and users, connect the database to a server, and connect the server to the client.

  • Undead Siege

    HTML, CSS, TypeScript

    Undead Siege is a 2D top-down shooter Game taking place in a post-apocalyptic zombie world. The game employs sprite-based animations, tile-based maps, collision detection, basic AI, top-down movement, and efficient memory management, as well as other basic techniques commonly used in top-down games. Undead Siege was developed using the Wolfie2D game engine, developed by Richard McKenna and the underlying rendering technology is DirectX.

  • World Happiness Predictor

    Python, Linear Regression, Random Forest, Support Vector Machine models

    This project presents a comprehensive analysis and prediction of world happiness scores, leveraging datasets from 2015 to 2019. Focusing on factors like GDP, Health, Freedom, Government Corruption, and Generosity, the algorithm explores their correlation with happiness through linear regression. It further employs Linear Regression, Random Forest, and Support Vector Machine models to predict happiness scores, offering insights into the predictive relationships between these key factors and the overall happiness of countries.
